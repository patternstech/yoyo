POST /api/v1/notes

Create a new note for a patient.

Body Parameters

patientId (number, required)
Patient identifier who owns this note (min: 1).
title (string, optional)
Optional title for the note (max: 250 chars).
noteText (string, required)
The main content of the note (max: 1000 chars).
recordingPath (string, optional)
Optional path to associated audio recording (max: 500 chars).

Sample Request 

{
  "patientId": 123,
  "title": "Follow-up Visit",
  "noteText": "Patient is responding well to treatment. Energy levels have improved significantly. Continue current medication regimen.",
  "recordingPath": "/recordings/patient-123-note-1.mp3"
}

Sample Response 

{
  "success": true,
  "id": 1,
  "validationErrors": null
}
Success
200 OK
Errors
400 Bad Request
500 Server Error
PUT /api/v1/notes

Update an existing note.

Body Parameters

id (number, required)
The unique identifier of the note to update (min: 1).
patientId (number, required)
Patient identifier who owns this note (min: 1).
title (string, optional)
Optional title for the note (max: 250 chars).
noteText (string, required)
The main content of the note (max: 1000 chars).
recordingPath (string, optional)
Optional path to associated audio recording (max: 500 chars).

Sample Request 

{
  "id": 1,
  "patientId": 123,
  "title": "Updated Follow-up Visit",
  "noteText": "Patient is responding exceptionally well to treatment. Energy levels have improved significantly. Recommend continuing current medication regimen with slight dosage adjustment.",
  "recordingPath": "/recordings/patient-123-note-1-updated.mp3"
}

Sample Response 

{
  "success": true,
  "id": 1,
  "validationErrors": null
}
Success
200 OK
Errors
400 Bad Request
404 Not Found
500 Server Error
DELETE /api/v1/notes/{id}?patientId={patientId}

patientId (number, query, required)
